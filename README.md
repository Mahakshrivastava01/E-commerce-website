# Dummy-Clothing-Store
This site shows a store of cloth created using HTML, CSS. It is a simple and responsive clothing store website built using HTML and CSS, showcasing products using modern CSS Grid and Card layout.

🚀 Features:
🛍️ Product listing using card design
📐 Responsive layout with CSS Grid
🎨 Clean and modern UI
📱 Mobile-friendly design
🖼️ Product images with details (name, price, cart, etc.)

🛠️ Tech Stack:
HTML5
CSS3 (Grid & Flexbox)
